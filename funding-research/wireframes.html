Indigenous Population Explorer

National CSD/DA atlas with Census identity, socioeconomics, language, funding, and predictive gap analysis
What We're Building
A national Indigenous socioeconomic atlas

Interactive GIS dashboard mapping ~5,000 Census Subdivisions across all of Canada with ~56,000 Dissemination Areas for city-level zoom. Same architecture as the Classroom Complexity dashboard (Leaflet + TRACE CSS + config-driven). Alberta default view with province switcher.

The "How many Blackfoot in Toronto" question
Census Question 27 asks "Which First Nation or Indian band?" — the data EXISTS. But StatCan publishes band membership only at provincial level (Table 98-10-0267), not per-CSD. The CSD-level cross-tab requires a custom tabulation request (~$500-2,000, 4-8 weeks). The drill-down panel shows provincial band membership data now, with a note about the custom tab for exact city-level numbers.

Fact Tables
4 fact tables + 1 predictive
fact_csd_census~5,000 rows · 25+ columns · one row per CSD
ColumnTypeDescription
csd_idVARCHAR(7)StatCan CSD code (FK to dim_csd)
total_popINTEGERTotal population
indigenous_popINTEGERAboriginal identity population
first_nations_popINTEGERFirst Nations identity
fn_on_reserveINTEGERFN on-reserve
fn_off_reserveINTEGERFN off-reserve
metis_popINTEGERMetis identity
inuit_popINTEGERInuit identity
registered_indianINTEGERRegistered/Treaty Indian status
indigenous_shareDECIMAL(6,4)indigenous_pop / total_pop
poverty_rate_indigDECIMAL(5,2)MBM poverty rate, Indigenous
poverty_rate_totalDECIMAL(5,2)MBM poverty rate, total
median_income_indigINTEGERMedian total income, Indigenous
median_income_totalINTEGERMedian total income, total
govt_transfer_shareDECIMAL(5,2)Govt transfers as % of income (Indigenous)
no_certificate_shareDECIMAL(5,2)% Indigenous without high school
postsec_shareDECIMAL(5,2)% Indigenous with post-secondary
core_housing_needDECIMAL(5,2)Core housing need rate, Indigenous
employment_rate_indigDECIMAL(5,2)Employment rate, Indigenous
indigenous_lang_shareDECIMAL(5,2)% with Indigenous mother tongue
fact_da_census~56,000 rows · 10 columns · one row per DA
ColumnTypeDescription
da_idVARCHAR(8)StatCan DA code
csd_idVARCHAR(7)Parent CSD (FK)
total_popINTEGERDA total population
indigenous_popINTEGERIndigenous identity
indigenous_shareDECIMAL(6,4)Share
first_nations_popINTEGERFN identity
metis_popINTEGERMetis identity
inuit_popINTEGERInuit identity
fact_band_membership~2,000 rows · band × province · on/off reserve
ColumnTypeDescription
band_idINTEGERBand number (FK to dim_band)
provinceVARCHAR(2)Province where members live
members_totalINTEGERTotal members in this province
members_on_reserveINTEGERLiving on-reserve
members_off_reserveINTEGERLiving off-reserve

Note: This is at PROVINCIAL level (Table 98-10-0267). CSD-level band membership requires a StatCan custom tabulation (~$500-2K).

fact_csd_predicted~5,000 rows · funding gap model output
ColumnTypeDescription
csd_idVARCHAR(7)FK to dim_csd
need_scoreDECIMAL(5,2)Composite 0-100 (poverty + housing + education + income + remoteness)
predicted_funding_pcDECIMAL(10,2)Model prediction: expected $/capita given Census indicators
actual_funding_pcDECIMAL(10,2)Actual funding per capita (from fact_csd_funding)
funding_gapDECIMAL(10,2)actual - predicted (negative = underfunded)

OCAP framing: "given the pattern of allocation, which communities are anomalously under-served" — not "how much Nations deserve."
